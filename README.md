访客： 可浏览网站首页的家政公司展示、服务列表、网站公告及自然风光等内容；使用搜索功能查找特定家政公司或服务信息；通过留言反馈模块提交建议或问题；导航栏设计简洁清晰，便于快速了解平台功能和服务内容。
